# Modern academic GitHub Pages site

This version has a more refined academic design:
- serif headings
- soft earth-tone palette
- polished hero section
- project and research cards
- cleaner publication/contact blocks

## Publish steps
1. Create a public GitHub repository, ideally named `YOUR-USERNAME.github.io`
2. Upload `index.html` and `styles.css` to the repository root
3. Commit the files
4. In GitHub go to **Settings → Pages**
5. Choose **Deploy from a branch**
6. Select `main` and `/root`
7. Save

## Replace these placeholders
- `your.email@lmu.de`
- Google Scholar link
- ORCID link
- CV PDF link
